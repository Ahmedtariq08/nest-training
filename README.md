## Description

A project developed to implement the authentication and authorization logic with NestJS. 
Functionality includes register, login, logout and refresh token. JWT based authentication with refresh tokens.
Password hashing along with auth gaurds for routes, passport strategies for access and refresh tokens. PostgreSQL with Prisma client used as database.


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test
